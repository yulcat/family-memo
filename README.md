# 🏠 장위동 가족 메모장

가족끼리 간단한 메모를 남기고 공유할 수 있는 웹 앱입니다.

## ✨ 기능

- 📝 메모 작성/수정/삭제
- ☑️ **체크리스트 모드** - 장보기 목록, 할일 관리
- 📌 **핀/고정** - 중요한 메모 상단 고정
- 🎨 **색상 라벨** - 메모를 색상으로 분류 (빨강/노랑/초록/파랑/보라)
- 👨‍👩‍👧 가족 구성원별 작성자 선택 (사장님, 사모님, 햄집사)
- 🔄 30초마다 자동 새로고침
- 📱 모바일 반응형 디자인
- 📊 체크리스트 진행률 표시

## 🚀 실행 방법

```bash
# 의존성 설치
npm install

# 서버 실행
npm start
```

서버가 시작되면 http://localhost:3456 에서 접속할 수 있습니다.

## 🔧 환경 변수

| 변수 | 설명 | 기본값 |
|------|------|--------|
| `PORT` | 서버 포트 | 3456 |

## 📁 구조

```
family-memo/
├── server.js        # Express 서버
├── public/          # 프론트엔드 정적 파일
│   ├── index.html
│   ├── style.css
│   └── app.js
├── family-memo.db   # SQLite 데이터베이스 (gitignore)
└── package.json
```

## 🔒 보안

- 데이터베이스 파일(`*.db`)은 `.gitignore`에 포함되어 GitHub에 업로드되지 않습니다.
- 메모 내용은 로컬에만 저장됩니다.

## 💡 사용 팁

- **Ctrl + Enter**: 메모 빠른 저장
- 작성자를 선택한 후 메모를 작성하세요
- 다른 기기에서도 같은 네트워크라면 `http://[서버IP]:3456`으로 접속 가능

## 🎨 색상 라벨 의미

| 색상 | 의미 | 사용 예시 |
|------|------|----------|
| 🔴 빨강 | 긴급/중요 | 오늘까지 해야 할 일, 급한 연락 |
